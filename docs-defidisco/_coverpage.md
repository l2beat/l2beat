# DeFiDisco

> DeFi Protocol Dynamic Analysis by DeFiScan

DeFiDisco is an essential part of the V2 of the DeFiScan framework; its completeness includes advanced automation, more efficient reviewing, and continuous monitoring capabilities, altogether enabling the application of our new framework to the wide selection of existing DeFi protocols swiftly. The tool is an enhancement of L2Beat’s [discovery (”Disco”) and continuous monitoring mono-repository](https://github.com/l2beat/l2beat).
