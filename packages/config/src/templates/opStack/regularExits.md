The user initiates the withdrawal by submitting a regular transaction on this chain. When a state root containing such transaction is settled, the funds become available for withdrawal on L1 after {{disputeGameFinalityDelaySeconds}}. Withdrawal inclusion can be proven before state root settlement, but a {{proofMaturityDelaySeconds}} period has to pass before it becomes actionable. The process of state root settlement takes a challenge period of at least {{challengePeriod}} to complete. Finally the user submits an L1 transaction to claim the funds. This transaction requires a merkle proof.
