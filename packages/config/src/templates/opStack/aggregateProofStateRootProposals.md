State roots are proposed by calling `DisputeGameFactory.create` with the AggregateVerifier game type{{bondNote}}. Each proposal must include an initial proof (TEE attestation or ZK proof) over the range of {{blockInterval}} L2 blocks split into sub-ranges of {{intermediateBlockInterval}} blocks. With a single proof, the game resolves after {{slowFinalization}}; if both proof arms commit, the window collapses to {{fastFinalization}}.
