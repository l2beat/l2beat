These steps reproduce the current 1,166-felt SHARP outer bootloader on Linux. The catalog identifier is the standard Cairo Pedersen program hash, which also commits to the entrypoint and builtin header. Ethereum does not store this derived hash: the immutable `CairoBootloaderProgram` contract pins the compiled instruction array directly.

Dependencies: Git, Bazelisk configured to use Bazel 7.4.1, JDK 21, GMP development headers, Python 3, and Foundry `cast`.

1. Check out the exact public source revision:

```bash
git clone https://github.com/starkware-libs/cairo-lang.git
cd cairo-lang
git checkout 56407b69f3f19f69302a8623baa8c5f71f967eed
```

The Cairo compiler version at this revision is `0.15.0a0`. The entrypoint is `src/starkware/cairo/bootloaders/bootloader/bootloader.cairo`.

2. Build the official proof-mode target:

```bash
USE_BAZEL_VERSION=7.4.1 bazelisk build \
  //src/starkware/cairo/bootloaders/bootloader:bootloader_program
```

The compiled program is written to `build/bazelbin/src/starkware/cairo/bootloaders/bootloader/bootloader_compiled.json`. The target uses `--proof_mode --debug_info_with_source`.

3. Fetch the instruction array pinned by the immutable Ethereum contract. Set `ETHEREUM_RPC_URL` to an archive or current Ethereum RPC endpoint first.

```bash
cast call 0x24105e6697AdD9B4B1BDE04079a91BDFCCa24A47 \
  'getCompiledProgram()(uint256[1166])' \
  --rpc-url "$ETHEREUM_RPC_URL" \
  --json > /tmp/sharp_outer_bootloader_onchain.json
```

4. Compare every rebuilt felt with the onchain array:

```bash
python3 - <<'PY'
import json
from pathlib import Path

compiled = json.loads(
    Path(
        "build/bazelbin/src/starkware/cairo/bootloaders/bootloader/bootloader_compiled.json"
    ).read_text()
)
onchain = json.loads(
    Path("/tmp/sharp_outer_bootloader_onchain.json").read_text()
)[0]
rebuilt = [int(value, 0) if isinstance(value, str) else value for value in compiled["data"]]

assert len(rebuilt) == 1166
assert len(onchain) == 1166
assert rebuilt == onchain
print("All 1166 compiled felts match the onchain program.")
PY
```

5. Compute the standard Cairo Pedersen program hash used as the catalog identifier:

```bash
USE_BAZEL_VERSION=7.4.1 bazelisk run \
  //src/starkware/cairo/bootloaders:cairo_hash_program_exe -- \
  --program "$PWD/build/bazelbin/src/starkware/cairo/bootloaders/bootloader/bootloader_compiled.json" \
  --program-hash-function pedersen
```

The command should print:

```text
0x79427e32341aeae95a32eb1ec1c8b43f014aaafa2781d1102635a0f81598f28
```

Its decimal representation, used as the catalog key, is `3427958597398434235135013788958741576989752718219267963615783564775551242024`.
