1. Checkout the pinned Starknet sequencer source:
   ```bash
   git clone https://github.com/starkware-libs/sequencer.git
   cd sequencer
   git checkout APOLLO-0.14.3-RC.11
   test "$(git rev-parse HEAD)" = "5114457ad4b5d6d1764b520dfa40b9e826f48854"
   ```
2. Inspect `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/os_config/os_config.cairo`. The V4 commitment uses Cairo's `encode_felt252_data_and_calc_blake_hash` over the ordered preimage:
   - the felt encoding of `StarknetOsConfig4`,
   - the felt encoding of the mainnet chain ID `SN_MAIN`,
   - the STRK fee-token address `0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d`.

   The pinned deployment configuration supplies `SN_MAIN` in `deployments/sequencer/configs/overlays/hybrid/mainnet/common.yaml` and the STRK address in `deployments/sequencer/configs/overlays/hybrid/common/common.yaml`. The optional state-diff-encryption public-key hash is zero for the current mainnet configuration, so the Cairo implementation does not append it. The equivalent Rust implementation is `OsChainInfo::compute_os_config_hash` in `crates/starknet_api/src/core.rs`.
3. Run this standard-library Python reproduction of the Cairo Blake2s-Felt252 encoding:
   ```bash
   python3 - <<'PY'
   import hashlib

   FIELD_PRIME = 2**251 + 17 * 2**192 + 1
   SMALL_THRESHOLD = 2**63
   preimage = [
       int.from_bytes(b"StarknetOsConfig4", "big"),
       int.from_bytes(b"SN_MAIN", "big"),
       int("04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d", 16),
   ]

   words = []
   for felt in preimage:
       encoded = felt.to_bytes(32, "big")
       if felt < SMALL_THRESHOLD:
           words.extend([
               int.from_bytes(encoded[24:28], "big"),
               int.from_bytes(encoded[28:32], "big"),
           ])
       else:
           limbs = [
               int.from_bytes(encoded[offset:offset + 4], "big")
               for offset in range(0, 32, 4)
           ]
           limbs[0] |= 1 << 31
           words.extend(limbs)

   encoded_preimage = b"".join(word.to_bytes(4, "little") for word in words)
   digest = hashlib.blake2s(encoded_preimage).digest()
   config_hash = int.from_bytes(digest, "little") % FIELD_PRIME

   expected = 2579130946496422157802313572919622021390761807038780433165936715591440018810
   assert config_hash == expected
   print(f"0x{config_hash:064x}")
   print(config_hash)
   PY
   ```
4. The output should equal the current onchain `Starknet.configHash` in hexadecimal and decimal form:
   ```text
   0x05b3bc83b3a47231aa5f15e73309ef7e09e087bd4ace767656fe5d713f6e2d7a
   2579130946496422157802313572919622021390761807038780433165936715591440018810
   ```
