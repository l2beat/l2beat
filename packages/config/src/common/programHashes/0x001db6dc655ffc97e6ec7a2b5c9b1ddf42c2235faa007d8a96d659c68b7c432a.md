Prepare:

1. Install cargo make: `cargo install --debug --locked cargo-make`
2. Install sp1 toolchain: `curl -L https://sp1up.succinct.xyz/ | bash`, then `sp1up`
3. Install docker [https://docs.docker.com/get-started/get-docker/](https://docs.docker.com/get-started/get-docker/)

Verify:

1. Checkout the correct tag in [mantle-xyz/op-succinct](https://github.com/mantle-xyz/op-succinct/tree/v2.2.4-mainnet.4) repo:  `git checkout v2.2.4-mainnet.4` . Commit hash should be  `614ab563542d5ce4c027de71765ecb4388c233af`.
2. Make sure docker is running by running  `docker ps`
3. From the  `op-succinct` dir:  `cargo run --bin config --release` to build the SP1 programs and generate and print verification key hashes. The `Aggregation Verification Key Hash` printed is this program hash.
