Even though the program is compiled in docker for reproducibility reasons, it gives the correct results only on linux OS. Steps below were done on Ubuntu 22.04 OS. The steps below consume ~95 GiB disk space.

1. Install docker [https://docs.docker.com/get-started/get-docker/](https://docs.docker.com/get-started/get-docker/).
2. Checkout the correct branch in [nitro](https://github.com/OffchainLabs/nitro) repo:  `git checkout consensus-v61` . Commit hash should be  `dfab1dea5c8d2c5089eb26d22ad6239bb78cfa5d`.
3. Update git submodules `git submodule update --init --recursive --force`.
4. Fix a dependency on a private repo in `Dockerfile`: replace line `RUN --mount=type=secret,id=gh_token,required=false ./download-machine.sh consensus-v60-rc.3 0xeadc910d36a0eab43a3dd7acb1a3513885e333785689c7cf750b9949c5bb7227 OffchainLabs/nitro-private` with `RUN ./download-machine.sh consensus-v60-rc.5 0x7a9e6a77354888257a9989ce0b6bb39df5fedf222d453932933fdf7a489cbb57`.
4. Generate wasm module root in docker: `docker buildx build --target nitro-node-dev -t nitro-node-dev .`.
5. Export the value from the docker: `docker run --rm --entrypoint cat nitro-node-dev /home/user/target/machines/latest/module-root.txt`.
