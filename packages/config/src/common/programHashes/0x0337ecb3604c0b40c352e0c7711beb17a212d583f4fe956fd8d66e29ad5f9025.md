1. Install [just](https://just.systems).
2. Install docker [https://docs.docker.com/get-started/get-docker/](https://docs.docker.com/get-started/get-docker/) and make sure it is running.
3. Check out the correct commit of the [optimism](https://github.com/ethereum-optimism/optimism) repo (tag `kona-client/v1.6.0-rc.2`): `git checkout d7cea91bc2f555a76b7720bf9c32f46c0b856119`.
4. Go to the kona dir `cd kona` and run `just build-cannon-prestate`. This script regenerates the kona absolute prestate and creates `kona/prestates/<hash>.bin.gz`, where `<hash>` is the resulting absolute prestate.
