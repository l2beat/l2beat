Regeneration steps are based on [this guide](https://github.com/fluentlabs-xyz/fluent-stf/blob/v1.0.0/README.md). The process is reproducible on a Linux machnie.

1. Install prerequesits: `docker`, `python3`, `git`, `jq`.
2. Checkout correct branch in https://github.com/fluentlabs-xyz/fluent-stf/tree/v1.0.4: `git checkout v1.0.4`. Commit hash should be `e2cd6942c1dd9200834abd9e816e38b83d1caa51`.
3. Build Nitro program for the mainnet within docker: `make build-nitro-validator-docker NETWORK=mainnet`. This command will create `nitro-validator-mainnet.vkey` file with the program hash string.
