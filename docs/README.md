# DeFiScan V2

DeFiScan V2 is the second generation of the DeFiScan framework. It adds advanced automation, efficient reviewing, and continuous monitoring so the DeFi Collective's security framework can be applied to a wide range of existing DeFi protocols. The tool is built on top of L2BEAT's [discovery and continuous monitoring monorepo](https://github.com/l2beat/l2beat).
