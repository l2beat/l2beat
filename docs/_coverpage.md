# DeFiScan V2

> DeFi Protocol Dynamic Analysis by DeFiScan

DeFiScan V2 is the second version of the DeFiScan framework; it includes advanced automation, more efficient reviewing, and continuous monitoring capabilities, altogether enabling the application of our new framework to the wide selection of existing DeFi protocols swiftly. The tool is an enhancement of L2Beat's [discovery and continuous monitoring mono-repository](https://github.com/l2beat/l2beat).
